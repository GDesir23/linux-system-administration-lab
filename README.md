# Linux-system-administration-lab
Built and configured an Ubuntu Server 26.04 LTS virtual machine using VirtualBox on an Apple Silicon Mac.
Technologies Used

* Ubuntu Server 26.04 LTS
* VirtualBox 7.2.8
* Linux CLI
* OpenSSH

Objectives

* Deploy a Linux server
* Configure storage and networking
* Create administrator account
* Enable remote management
* Perform basic system administration tasks

Skills Demonstrated

* Linux installation and configuration
* User account management
* File system management
* SSH configuration
* System monitoring
* Command-line administration
